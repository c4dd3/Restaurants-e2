Arquitectura — Etapa 2

Tres microservicios (api · auth · search) detrás de un balanceador Nginx, un solo motor de BD por run (postgres o mongo), arquitectura hexagonal con sub-DAOs por adapter. Cada diagrama vive en su propia página para que se lea cómodo.
